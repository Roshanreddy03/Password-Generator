# Password-Generator
⚙️ Program Workflow:

🔹 First, the program asks the user how many letters they want in the password.

🔹 Then, it asks for the number of numbers.

🔹 Next, it asks for the number of symbols.

🔹 The program generates a random password based on the given requirements.

🔹 Finally, it shuffles the characters to create a strong and secure password .

--------------------------------------------------------------------------------------------------------

 🔁 Replay Option:

✅ If the user selects Yes, the program asks for new input values again.

❌ If the user selects No, the program terminates.
